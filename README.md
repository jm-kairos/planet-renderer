Planet Renderer
An OpenGL-based 3D planet renderer built in C++14 that generates and renders spherical terrain using tessellation shaders.
Features
•	Spherical Cube Geometry — Generates planet-like spheres by projecting a subdivided cube onto a sphere, with configurable radius and tessellation resolution
•	GPU Tessellation — Uses tessellation control and evaluation shaders for dynamic level-of-detail terrain subdivision
•	Heightmap-based Terrain — Supports displacement mapping via heightmap textures (e.g., Iceland heightmap) for realistic surface detail
•	First-Person Camera — Fly-through camera system with WASD movement, mouse look, and shift for speed boost
•	Dynamic Lighting — Orbiting light source around the planet surface
